# Hanoi Business Travel Guide

This is a static HTML travel guide page ready for GitHub Pages hosting.

## Files
- `index.html` — Main website file
- `.nojekyll` — Keeps GitHub Pages simple for static hosting

## How to host on GitHub Pages

1. Create a new GitHub repository, for example:
   `hanoi-travel-guide`

2. Upload these files to the repository:
   - `index.html`
   - `.nojekyll`
   - `README.md`

3. Go to:
   Repository Settings → Pages

4. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ root**

5. Click **Save**.

6. Your page will be live at:
   `https://YOUR-GITHUB-USERNAME.github.io/hanoi-travel-guide/`

## Notes
- All map buttons open Google Maps.
- The VND to INR converter uses an online exchange-rate API, so it needs internet access.
- No backend is required.
